Restricting Rows

A DataFrame can have it's rows restricted in one of two ways:
  • By index using the .iloc object
  • By a condition on one of the fields

By Index

A slice of the rows can be selected by their index number using the .iloc object
For Example
To return lines 3,4 and 5 you could use: data.iloc[3:6]
Note: Python never includes the end of the range

To return a single row, say row 8, either of the following could be used (they return different structures):
  • data.iloc[8:9]
  • data.iloc[9]
Test them in the Python file to investigate.

By Condition

The syntax for this is slightly unusual.
A condition is created, say the order quantities exactly equal to 10: data['Qty'] == 10
This creates a list of row number. This condition is then inserted into square brackets after the DataFrame name.

For Example:
data[data['Qty'] == 10]
This then return just the orders where the quantity is equal to 10.
