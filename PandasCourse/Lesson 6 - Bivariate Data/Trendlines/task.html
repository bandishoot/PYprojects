Adding a Trendline

Adding a trendline is a little more involved than the scope of this course. However, when the pandas library was installed, python also installed the numpy library as a prerequisite, which has some methods that can be used.

A regression must be performed on the data points to create a linear trendline and then it is added to the same plot.
