Histograms

Histograms are a great way to investigate the distribution of univariate numerical data. You have been given data on the rate rates of 24 students before and after exercise.

To plot a histogram simply change the kind argument to "hist":
data.plot(kind="hist")

This plots both plots on top of each other with solid block colours. To make it easier to see both distributions try:

  • To change the transparency: data.plot(kind="hist", alpha=0.5)
  • Or, plot two graphs: data.plot(kind="hist", subplots = True)
To alter the number of columns add the bins argument to the .plot method().
data.plot(kind="hist", subplots = True, bins=8)
