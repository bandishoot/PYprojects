Sorting the DataFrame

At some point in all data analysis it is necessary to sort the data.
Pandas uses the .sort_values() method to do this for you.

The syntax for the .sort_values() method is:
.sort_values([column_names], ascending = True, inplace=False)

NOTE: the final two parameters are optional, I have displayed their default values if they are omitted.

If we wanted to sort our shop orders in ascending order by price we could do it in one of two ways:
data.sort_values(['Price'], ascending = True)
or,
data.sort_values(['Price'])


The inplace parameter when set to True, will persist the sorted order to the dataframe.
