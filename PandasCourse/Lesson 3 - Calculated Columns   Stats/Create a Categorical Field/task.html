Creating a Categorical Field


This is for reference only, and is beyond the intention of this course.

Sometimes it can be necessary to give create a categorical field as a classifier. An example that we've seen so far to to classify people on their BMI.

The code in the python file creates a new field BMIClass that has one of 4 options: underweight, normal, overweight, obese.
The apply method needs to be run here in conjunction with a user defined function for the decision making.
