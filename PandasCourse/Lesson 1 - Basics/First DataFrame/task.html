Creating a DataFrame

A DataFrame is a table of values made up of fields in columns and rows of data. It is a bit like a table in Excel.

The DataFrame is the fundamental object in this course and we'll be learning how to manipulate it, do calculations on it and draw charts.

To create your first DataFrame, you'll need to import the pandas library. At the top of your python file, type:
import pandas as pd

This will import the library and shorten it's name to pd, so we don't have to type pandas all the time.

Let's say we had some data on Tim, Jane and Alfie, such as their height, weight and favourite colour. we could set up a DataFrame using dictionaries with the columns as the keys and lists of data points as the value.

For Example:
dataDict = {'name':['Tim','Jane','Alfie'], 'height':[1.73, 1.68, 1.70], 'weight':[68, 60, 80],'colour':['red','blue','yellow']}

We can then use this dictionary to build a DataFrame:

data = pd.DataFrame(dataDict)
