Modify The Plot

Hopefully in the last section you managed to plot the chart below:

This isn't ideal as a final output.


Charts require:
  • Title
  • Axis Titles
  • Axis Labels

Firstly, we want our subjects to be the x-axis labels. So we set the index of the DataFrame to be the subjects column:
data.index=data["subject"]

Run the program and plot the chart again, however, you'll find the labels are rotated 90 degrees and difficult to read. This is set during the initial .plot() method, add rot=0 to keep them horizontal.

Whilst altering this line, you can also alter the colours of the chart, as a list (an item for each category) and also add a title.
data.plot(kind="bar", color=["blue","red"],rot=0, title="Favorite Subjects")

Lastly let's add some axis titles:
plt.xlabel('Subject')
plt.ylabel('Frequency')
