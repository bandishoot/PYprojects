Pie Charts

The same categorical data can be plotted in a pie chart.
To plot a pie chart, we simply alter the kind argument in the .plot() method. An argument to enable subplots must be used in this case because we have two pie charts.

data.plot(kind="pie", subplots=True)

Run the program; the legend is not required in this case, as the slices of pie are already labelled. To remove the legend set it to false in the .plot() method.
data.plot(kind="pie", subplots=True, legend=False)

Titles and colours (one for each category) can be added again:
data.plot(kind="pie", subplots=True, legend=False, colors=['red','pink','orange','yellow','blue','green'], title ="Favorite Subjects")
