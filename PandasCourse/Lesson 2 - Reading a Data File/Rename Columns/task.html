Changing the Column Names

It might be necessary to alter the names of columns, there can be many reason for this:
  1. There is no header in the CSV file.
  2. The current names have spaces in (this tends to cause issues during some analysis.
  3. The naming convention is inconsistent.
  4. the names are meaningful.
There are other reasons, but these are common. Please note, that your columns names should therefore try to match the criteria above.

The columns names is a pandas DataFrame are stores as a list under the columns property for the specific DataFrame. It can be easily accessed and edited.

From the example in the current programme, try running: data.columns=['name','email','dob','height','weight','smoker']
