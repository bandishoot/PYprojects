Read a CSV with No Header

Occasionally a CSV file will contain just raw data and will require all the rows to be read as data into the DataFrame, giving you no columns names (we'll handle that shortly).

Should these situation occur, simply adding header=None to the read_csv method will ensure the pandas library doesn't try to assign column names.

For Example
try running the following line
data = pd.read_csv("healthNoHeader.csv", header=None)
